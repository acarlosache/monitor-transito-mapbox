# 🚦 Monitor de Trânsito - Mapbox

Este é um projeto de monitoramento de trânsito desenvolvido para visualizar condições de tráfego em tempo real utilizando mapas interativos.

## 🛠️ Tecnologias Utilizadas
* **HTML5 / CSS3**: Estrutura e estilização da interface.
* **JavaScript (ES6+)**: Lógica de integração e manipulação de dados.
* **Mapbox GL JS**: Biblioteca de mapas para renderização de camadas de tráfego.

## 🚀 Funcionalidades
* Visualização de mapas de alta performance.
* Camada de tráfego em tempo real.
* Marcadores personalizados (como o ícone de sinalização).

## 📂 Como rodar o projeto
1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador (recomenda-se usar a extensão Live Server no VS Code).
3. Insira sua chave de API do Mapbox no arquivo `script.js`.
